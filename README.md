Проект реализует метод, который по трём длинам сторон треугольника: 
Определяет его тип: равносторонний, равнобедренный, разносторонний или не треугольник. 
Вычисляет координаты трёх вершин, отмасштабированные для отображения в области 100×100 пикселей. 
Ведёт детальное логирование всех запросов (успешных и ошибочных).
